Mini Banco em Java

Sistema bancário em Java executado no terminal, desenvolvido com foco em lógica de negócio, programação orientada a objetos e uso de estruturas de dados.

Funcionalidades

Criação de contas com senha

Autenticação por ID

Depósito e saque com validações

Transferência entre contas

Extrato com histórico de transações por conta

Estrutura do Projeto

Main — Interface e menus

Banco — Regras do sistema

Conta — Modelo da conta bancária

Transacao — Registro das operações

Tecnologias

Java

HashMap e List

Programação Orientada a Objetos

Execução
javac *.java
java Main
